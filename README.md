# Реалізація обробки виключень та модульності в Python

Цей проект є частиною Лабораторної роботи №3, присвяченої вивченню механізмів обробки помилок, створенню власних ієрархій виключень, організації коду в пакети та документуванню публічного API.

## 📌 Опис проекту

Проект демонструє систему обробки математичних даних. Основний функціонал включає:
- Валідацію вхідних даних.
- Виконання безпечних математичних операцій (ділення, корінь).
- Роботу з файловою системою через контекстні менеджери.
- Роботу з ієрархією доменних помилок.
